## üíª Introduction to Proxmox VE

**Proxmox Virtual Environment (VE)** is an open-source Type-1 hypervisor used for managing virtual machines and containers.  
I learned about its architecture, features, and performed a installation on Dell server R710.

### üß© Steps Performed:
1. Downloaded **Proxmox VE ISO** and created a bootable USB.  
2. Installed Proxmox on a **Dell R710** server.  
3. Configured network bridge for VM connectivity.  
4. Accessed the **Proxmox Web GUI** for creating virtual machines.  
5. Created an Ubuntu Server VM for testing.  

> üß† Key Learning: Understood the concept of **Type 1 vs Type 2 hypervisors**, and how Proxmox allows managing multiple VMs from a centralized web interface.
